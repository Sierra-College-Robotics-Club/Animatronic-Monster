A project by Sierra College Robotics Club, demonstrating an interactive animatronic monster,
affectionately nicknamed "Gancho", and designed for Open Sauce 2026.

V1 uses various parts from other projects:
- The base PCB used is the [Glow Clock](https://github.com/Sierra-College-Robotics-Club/GlowClock/tree/main)
- The control panel uses the [STM32L0 i2c gpio expander](https://github.com/Patronics/stm32l0-i2c-gpio-expander) running on the [Patronics STM32DIP20 breakout board](https://github.com/Patronics/STM32DIP20-breakout).
